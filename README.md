# FileManager
### Dependencies
- Visual stusio / GCC
- VCPKG
- Cmake 
- SFML
---
### How build from source
1. Clone source 
    - `git clone ...`
2. Configure project
    - `mkdir build && cd build && cmake .. -DCMAKE_BUILD_TYPE=Release`
3. Build project
    - `cmake --build . --config release --parallel`
4. Install dependencies
    - ...
5. Execute 
    - Run file 'FileManager'
